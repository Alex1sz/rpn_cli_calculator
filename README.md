## How to Run
1. Install: `go get github.com/alex1sz/rpn_cli_calculator`
1. To Build & Run: `go build && ./rpn_cli_calculator`
1. To Run: `go run main.go` or `./rpn_cli_calculator`
1. Supply values to evaluate `./rpn_cli_calculator 2 3 +`
